Sharp Security from All Angles

Tiffany Ford

This workshop is a scaled-down version of the Secure From Scratch presentation I attended at Defcon 32 in August 2024. We will look at a .NET Core 8 / Angular 17 application and discuss security vulnerabilities, learn how attackers think about exploitation and how you can think securely about your applications.

Attendees who want to work along with me during the workshop will need to get the application running on their laptop before the session. Instructions available in the GitHub for the test application.
